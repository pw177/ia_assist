Проект: ИИ-ассистент для компании по изготовлению и установке заборов

Описание проекта
ИИ-ассистент для компании по изготовлению и установке заборов — это интеллектуальная система на базе больших языковых моделей, предназначенная для автоматизации первичной коммуникации с потенциальными и существующими клиентами.

Ассистент способен:
Вести осмысленный диалог на естественном языке
Консультировать по ассортименту, ценам и условиям монтажа
Квалифицировать лидов и собирать контактные данные
Записывать клиентов на замер/встречу с автоматической синхронизацией в Google Календарь

Целевая аудитория: владельцы загородной недвижимости, строительные компании, управляющие ТСЖ, частные застройщики.

Функциональные возможности:
💬 Осмысленное ведение диалога
Пример диалога:
Клиент: Здравствуйте, сколько стоит забор из профнастила на 20 метров?
ИИ: Добрый день! Для расчёта стоимости мне нужно уточнить несколько деталей:
     1. Какой высоты планируете забор? (1.5м, 1.8м, 2м)
     2. Нужны ли ворота и калитка?
     3. Есть ли уже подготовленные столбы или требуется монтаж "под ключ"?

📚 Ответы на вопросы по базе знаний:
База знаний включает:
  - 🏗️ Продукция: типы заборов (профнастил, евроштакетник, сетка-рабица, 3D-секции)
  - 💵 Прайс-лист: стоимость материалов, работ, дополнительных опций
  - 🚚 Логистика: зоны доставки, сроки, условия бесплатного выезда замерщика
  - 🔧 Монтаж: этапы работ, гарантийные обязательства, требования к участку
  - ❓ FAQ: ответы на частые вопросы (оплата, договор, демонтаж старого забора)
